# 이메일 템플릿 (email_template.pyw)

변수 치환이 가능한 이메일 템플릿을 관리하고 빠르게 작성하는 도구.

---

## 기능

- 템플릿 추가/편집/삭제
- `{{변수명}}` 형식 변수 자동 감지
- 변수 입력 후 치환된 본문 미리보기
- 완성된 이메일 클립보드 복사

---

## 데이터

저장 파일: `email_templates.json`

```json
{
  "templates": [
    {
      "id": 1,
      "name": "템플릿 이름",
      "subject": "메일 제목 {{수신자}}님께",
      "body": "안녕하세요, {{수신자}}님.\n\n{{내용}}\n\n감사합니다."
    }
  ]
}
```

> `email_templates.json`은 사용자 데이터로 `.gitignore` 제외.
