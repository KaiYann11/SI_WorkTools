# 이메일 템플릿 (email_template.pyw)

변수 치환이 가능한 이메일 템플릿을 관리하고 빠르게 작성하는 도구.

---

## 기능

- 템플릿 추가/편집/삭제
- `{{변수명}}` 형식 변수 자동 감지
- 변수 입력 후 치환된 본문 미리보기
- 완성된 이메일 클립보드 복사

---

## 데이터

저장 파일: `email_templates.json`

```json
{
  "templates": [
    {
      "id": 1,
      "name": "템플릿 이름",
      "subject": "메일 제목 {{수신자}}님께",
      "body": "안녕하세요, {{수신자}}님.\n\n{{내용}}\n\n감사합니다."
    }
  ]
}
```

> `email_templates.json`은 사용자 데이터로 `.gitignore` 제외.

---

## 변수 치환 규칙

- 패턴: `{{변수명}}` (중괄호 2개)
- 템플릿 저장/로드 시 변수 목록 자동 파싱
- 변수 입력 UI: 감지된 변수마다 Entry 위젯 자동 생성

---

## 수정 시 체크리스트

| 작업 | 확인 항목 |
|------|-----------|
| 변수 패턴 변경 | 파싱 정규식 + 미리보기 치환 로직 동시 수정 |
| 템플릿 필드 추가 | `_save()` + `_fill()` + JSON 스키마 |
| 클립보드 형식 변경 | `pyperclip` 또는 `tk.clipboard_*` 호출부 확인 |
